# BulkPDF

## Description
BulkPDF is a tool for filling PDF forms with data from Excel spreadsheets in bulk. It allows you to automatically populate multiple PDF forms with data from rows in an Excel file, saving you time when you need to generate many similar documents.

## Features
- Fill PDF forms with data from Excel spreadsheets
- Support for multiple PDF field types
- Batch processing of multiple records
- Command-line interface for automation
- GUI for easy configuration and testing

## Installation
1. Extract all files to a folder of your choice
2. Run BulkPDF.exe to start the application

## Usage
### GUI Mode
1. Launch BulkPDF.exe
2. Open your PDF form using the "Open PDF" button
3. Load your Excel data using the "Load Excel" button
4. Map the Excel columns to PDF fields
5. Configure output settings
6. Click "Generate" to create the filled PDFs

### Command Line Mode
```
BulkPDFConsole.exe [options]
```
Use BulkPDFConsole.exe --help for command line options.

## Examples
See the Examples folder for sample PDF forms and Excel files.

## Requirements
- Windows 7 or later
- .NET Framework 4.7.2 or later
- Microsoft Excel (for Excel data source)
